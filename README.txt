# EUR/USD Exchange Currency Pair - Time Series Forecasting

## Overview

This project focuses on time series forecasting for the EUR/USD exchange currency pair. It includes a dataset file, "EURUSD_18_12_2023.csv," and a Jupyter Notebook named "LSTM_time_series_forecasting_EUR_USD.ipynb." The project has been completed and is authored by Łukasz Golec.

## Project Contents

### Dataset
- The dataset file, "EURUSD_18_12_2023.csv," contains the necessary time series data for the analysis. The data is loaded from a cloud source, specifically OneDrive.

### Jupyter Notebook
- The Jupyter Notebook, "LSTM_time_series_forecasting_EUR_USD.ipynb," is the core of the project. It contains the code for time series forecasting using Long Short-Term Memory (LSTM) models. The notebook is designed to be run on Google Colab.
